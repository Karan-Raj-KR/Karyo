## 🚀 Overview

KĀRYO is a premium, cutting-edge digital agency portfolio built to showcase creative work, outline our meticulous process, and attract high-tier clients. Designed with a focus on immersive aesthetics, it features smooth scroll interactions, dynamic 3D elements, and a stark, beautiful dark-mode first design system.

## ✨ Features

- **Immersive 3D Interactions:** Custom cursors and subtle 3D elements powered by React Three Fiber.
- **Fluid Animations:** Seamless page transitions, scroll-linked animations, and micro-interactions using Framer Motion.
- **Modern UI Architecture:** Built on the robust foundation of React 19 and Tailwind CSS v4.
- **Fully Responsive Navigation:** A sleek, mobile-optimized navigation system with animated overlays.
- **Streamlined Booking Flow:** Built-in modal components for client onboarding and project booking.
- **Sections Included:**
  - Dynamic Hero with call-to-actions
  - Services Overview
  - Selected Work Showcase
  - Our Methodology / Process
  - Client Testimonials
  - Integrated Contact & Booking forms

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics:** [Three.js](https://threejs.org/) + [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction) + [Drei](https://github.com/pmndrs/drei)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** Fully typed in TypeScript for better developer experience and reliability.

## 💻 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- `npm` or any package manager of your choice

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Karan-Raj-KR/Karyo.git
   cd Karyo
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
   > The production-ready output will be located in the `dist` directory, easily deployable to platforms like Vercel, Netlify, or Cloudflare Pages.

## 🎨 Design Philosophy

KĀRYO embraces a "less is more" approach, focusing on bold typography, strategic whitespace, and meaningful motion. High-contrast elements ensure that the agency's actual work takes center stage, while the underlying framework provides a buttery-smooth user experience that implicitly communicates the agency's technical competence and attention to detail.
